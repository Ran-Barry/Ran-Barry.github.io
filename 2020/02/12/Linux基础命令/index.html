2.查看自己使用的Shell - ps

3.查看命令的说明和位置 - whatis/which/whereis

whatis

【功能】在whatis库中搜寻特定的命令

【语法】whatis COMMAND

whereis

【功能】 查找文件、手册页、命令等的相关位置

【语法】 whereis options argument

【常用选项】

【实例】

which

【功能】 查看可执行命令的路径

【语法】which COMMAND

4.清除屏幕上显示的内容 - clear

5.查看帮助文档 - man/info/help/apropos

6.查看系统和主机名 - uname/hostname
