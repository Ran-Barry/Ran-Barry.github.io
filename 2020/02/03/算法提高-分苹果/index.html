运用差分数组

代码很简洁，只计算一次就行
f[i] = d[i] - d[i-1]
d[2] = f[1]+d[0]+f[2] = d[1] + d[2] -d[1] = d[2]
数列第i项的值是可以用差分数组的前i项的和计算的，即d[i] = f[i]的前缀和
差分数组其实就相当于通过改变区间前段和末端与其他部分的差值，在最后进行累加的时候实行对整个区间的值的改变
